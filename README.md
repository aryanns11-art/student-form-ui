# 📝 Student Information Form (Python GUI)

A modern Student Information Form application built using **Python** and **CustomTkinter**.

This GUI-based application allows users to enter student details such as personal information, contact information, academic details, and address through a clean and user-friendly interface.

# 🚀 Features

* 👤 Enter Student Name
* 🎂 Enter Student Age
* 📧 Enter Email Address
* 📱 Enter Mobile Number
* 🚻 Select Gender
* 🎓 Select Course
* 📚 Select Academic Year
* 📅 Select Date of Birth
* 🏠 Enter Address
* ✅ Empty Field Validation
* 🖥️ Modern GUI using CustomTkinter
* 📋 Display Submitted Student Information

# 🛠️ Technologies Used

* Python
* CustomTkinter
